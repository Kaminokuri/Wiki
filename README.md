# Wiki

Dépôt GitHub de sauvegarde, de versionnement et de partage du contenu de mon wiki personnel propulsé par **Wiki.js**.

## À propos

Ce repository contient les pages, fichiers et versions de mon wiki synchronisé avec **Wiki.js**.

Il sert principalement à :

- conserver un historique clair des modifications ;
- sauvegarder le contenu du wiki ;
- partager la base de connaissances ;
- faciliter l’évolution de la documentation dans le temps.

## Site du wiki

Le wiki est consultable en ligne à cette adresse :

**https://wiki.matheo-fauvel.fr**

## Fonctionnement

Le wiki est édité principalement depuis **Wiki.js**, puis synchronisé avec ce dépôt GitHub.

### Synchronisation

- **Source principale** : Wiki.js
- **Versionnement** : Git / GitHub
- **Branche principale** : `main`

Selon la configuration, les changements peuvent être poussés automatiquement ou via une synchronisation manuelle.

## Contenu

Ce dépôt peut contenir :

- des pages de documentation en Markdown ;
- des fichiers liés au contenu du wiki ;
- une arborescence organisée par thèmes ;
- l’historique complet des modifications.

## Objectif du wiki

Ce wiki a pour but de centraliser mes connaissances, notes et documentations autour de différents sujets techniques, notamment :

- informatique ;
- systèmes et administration ;
- développement ;
- cybersécurité ;
- DevSecOps ;
- culture technique générale.

## Bonnes pratiques

Pour éviter les conflits :

- privilégier les modifications directement dans **Wiki.js** ;
- utiliser GitHub surtout pour le suivi, l’historique et le partage ;
- éviter d’éditer simultanément les mêmes pages à la fois dans GitHub et dans Wiki.js.

## Structure

L’organisation du dépôt suit la structure logique du wiki, avec une hiérarchie thématique et des fichiers Markdown.

Exemple :

```text
categorie/
├── index.md
├── sous-theme-1.md
├── sous-theme-2.md
└── sous-dossier/
    └── index.md
````

## Contribution

Ce dépôt est avant tout personnel, mais les suggestions, corrections et améliorations peuvent être proposées via :

* issues ;
* pull requests ;
* retours directs.

## Remarques

Ce repository ne contient pas l’application Wiki.js elle-même, mais uniquement le **contenu synchronisé** du wiki.
